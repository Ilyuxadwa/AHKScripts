## 📋 ОБЩАЯ ИНФОРМАЦИЯ

**Быстрый RP-отыгрыш для ФСБ с маскировкой под другие организации**

**🔥 ФУНКЦИИ:**
- **69 функций** для ФСБ на все возможные случаи для проведений арестов, наказывания неадекватов, проверок
- **Переключатель** Ctrl+LAlt для смены режима. (Обычный/Строй/Допрос + вербовка/Маскировка)
- **Команды** для всех функций.
- **Простые** сочетания клавиш для /c 060, отката и т.д.
- **Маскировка** под МО/МВД/ФСИН
- **Сохранение ID** игрока (Ввести id 1 раз в нужное поле, и далее оно подставляется всегда пока Вы не очистите его)
- **Full RP** вкл/выкл (отыгровки даже там, где не нужны)
- **Подсказки** в игре
- **Сокращения** для рации и не только
- **УК/КоАП/Уставы** (F10+1-7)

## 🛠️ НАЧАЛЬНАЯ НАСТРОЙКА

Для корректной работы скрипта необходимо настроить персональные данные. Откройте файл скрипта и укажите следующую информацию:

1. **Данные**:
   В разделе "Информация про пользователя" укажите ваши данные:
   ```ahk
   name := "Имя"         ; Укажите ваше имя
   surname := "Фамилия"  ; Укажите вашу фамилию
   department := "Отдел" ; Укажите ваш отдел
   rank := "Звание"      ; Укажите ваше звание
   alias := "Позывной"   ; Укажите ваш позывной
   tag := "ТЕГ"   ; Укажите ваш ТЕГ в рации

2. **Настройки**:
   В разделе "Пользовательская настройка скрипта" поставьте стандартные значения под себя:
   (true - включить, false - отключить)
   ```ahk

   keyModifier := "" ; Специальная клавиша для замены Numpad'a (Нажимаете эту клавишу + цифру на основной клавиатуре), если пусто - используется Numpad.
   full_rp := true   ; Отыгровки даже там, где не нужны (Если false, то останутся только обязательные)
   hotkeys := true   ; Включение и отключение горячих клавиш
   commands := true  ; Включение и отключение команд
   
Во время включенного ахк так-же можно изменять (клавиши снизу), но после перезапуска оно сбросится в отличии от этого варианта.

## ⚙️ НАСТРОЙКИ (Горячие клавиши)

| **Клавиша** | **Функция** | **Статус** |
|-------------|-------------|------------|
| **Ctrl + LAlt** | 🔄 Переключить режим | Обычный → Строй → Допрос/Вебовка → Маскировка |
| **Ctrl + P** | 🎭 Full RP вкл/выкл | `Фулл РП: true/false` |
| **Alt + K** | ⌨️ Горячие клавиши вкл/выкл | `Клавиши: true/false` |
| **Alt + C** | 💬 Команды вкл/выкл | `Команды: true/false` |
